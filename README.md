## Overview
This project presents an interactive Power BI dashboard designed to analyze the impact of smoking on health risks using patient data.

The dashboard focuses on how smoking status and duration influence organ health and chronic risk indicators, supporting better understanding through clear visual storytelling.

## What This Dashboard Shows
- Patient demographics including age, BMI, and gender
- Comparison of smoking status: never, former, and current smokers
- Organ health visualization for lungs, heart, liver, and kidneys
- Risk indicators related to hypertension and cholesterol
- Interactive filters for deeper analysis

## Tools Used
- Power BI
- Excel / CSV dataset
- Data cleaning and visualization techniques

## Key Insights
- Current smokers exhibit higher health risks across multiple organs
- Increased smoking duration correlates with higher chronic risk
- Visual analysis highlights the long-term benefits of non-smoking
